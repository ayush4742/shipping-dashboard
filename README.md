# 📦 Shipping Route Efficiency Dashboard

🚀 **Live App:** https://your-app-name.streamlit.app  
📊 **Project Type:** Data Analysis + Interactive Dashboard  

---

## 🎯 Objective

This project analyzes factory-to-customer shipping performance to identify:

- Fastest and slowest delivery routes  
- Regional bottlenecks  
- Shipping mode efficiency  
- Opportunities for logistics optimization  

---

## 🧠 Problem Statement

Nassau Candy Distributor lacked visibility into:

- Efficient vs inefficient routes  
- Delivery delays across regions  
- Impact of shipping methods  

This project provides a **data-driven solution** to improve logistics performance.

---

## ⚙️ Tech Stack

- 🐍 Python  
- 📊 Pandas, NumPy  
- 📈 Matplotlib, Seaborn  
- 🌐 Streamlit  
- 📍 Plotly (for map visualization)  

---

## 📊 Key Features

### ✅ Interactive Dashboard
- Real-time filtering (Region, Ship Mode, Lead Time)
- Dynamic KPI metrics

### 📍 US Heatmap
- State-wise delivery performance visualization

### 🏆 Route Analysis
- Top 10 fastest routes  
- Top 10 slowest routes  

### 🌍 Regional Insights
- Compare performance across regions  

### 🚚 Shipping Mode Analysis
- Evaluate delivery efficiency by shipping type  

---

## 📈 Key Insights

- Significant variation exists at the **route level**
- Regional performance is relatively **consistent**
- Some shipping modes show **unexpected inefficiencies**
- Identified multiple **high-delay routes** needing optimization  

---

## 🚀 Recommendations

- Optimize slow-performing routes  
- Improve logistics in bottleneck states  
- Re-evaluate expedited shipping processes  
- Use data-driven decision-making for route planning
